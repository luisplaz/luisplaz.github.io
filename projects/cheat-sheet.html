RUBY

Proper Syntax

  1. Returns are default it can be re stated but a ruby method will always return something by default.
  2. No semicolons are needed at the end of a line of code.
  3. Normally the use of capital letters is reserved for specific ocassions, use lowercase if uncertain
  4. Methods are mor straightforward than one might think sometime writing as one would think would work actually works!
